## Installation

Install packages individually based on your needs:

```bash
# Core utilities (usually included as dependency)
npm install @tooly/core

# Email functionality
npm install @tooly/resend

# Project management
npm install @tooly/linear
```

Or install multiple packages:

```bash
npm install @tooly/resend @tooly/linear
```

## Common Patterns

All packages follow consistent patterns for integration:

### Basic Usage
```typescript
import { PackageTools } from "@tooly/package-name";

// Initialize with API credentials
const tools = new PackageTools("your-api-key");

// Get tool definitions
const toolDefinitions = tools.getTools();

// Execute functions directly
const result = await tools.executeFunction("functionName", parameters);
```

### AI Framework Integration
```typescript
import { 
  createVercelAITools,
  createOpenAIFunctions, 
  createAnthropicTools 
} from "@tooly/package-name";

// AI SDK
const vercelTools = createVercelAITools("api-key");

// OpenAI
const { tools, executeFunction } = createOpenAIFunctions("api-key");

// Anthropic
const anthropicSetup = createAnthropicTools("api-key");
```
